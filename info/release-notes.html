02/jun/2009

  • Inauguramos uma nova aba, “Erros”, que leva às páginas de erro (módulo application). Em cada erro, clicar de novo na aba Blog leva ao erro seguinte.
  • O slogan, que era uma elemento h2, agora virou um em. Concentramos a força dos headings no #main_content.
  • Adicionamos novas folhas de estilo, para layout em outras mídias: dispositivos móveis, impressão e iPhone. No Be on the Net estático, elas estão vazias: não há CSS para esses meios. Mas quando o designer quiser fazer esse CSS em seu template, são essas as folhas que ele tem de usar.
  • Reordenamos as folhas de estilo. Adotamos um esquema de folhas que vai da mais genérica à mais específica, para facilitar ao designer a leitura: agora vem primeiro a tipografia (para todos os meios), depois a cor (para meios eletrônicos), depois os layouts (um para cada meio), e por último as exceções (para o IE).
  • Acrescentamos ao HTML as chamadas às novas folhas de estilo, para quando você quiser fazer CSS de layout para um dos meios citados anteriormente.
